# Projet Chattons Prospection

## Aperçu du projet
Ce projet vise à créer un site de publicité pour chatons "très réaliste", réalisé entièrement en JavaScript. Le but est de prospecter ou de présenter des chatons de manière attrayante et professionnelle.

### Technologies principales
- **Langage :** JavaScript (Vanilla ou Framework à définir)
- **Style :** CSS moderne pour un aspect réaliste et poli.

## Construction et Exécution
*Ce projet est en phase initiale de conception. Les commandes suivantes sont des placeholders.*

- **Installation :** `npm install` (si un package.json est ajouté)
- **Lancement (Dev) :** Ouvrir `index.html` ou utiliser un serveur local (ex: `live-server`).
- **Tests :** À définir.

## Conventions de Développement
- Prioriser un rendu visuel de haute qualité (réaliste).
- Code JavaScript propre et modulaire.
- Utilisation de ressources visuelles (images/gradients) pour renforcer l'aspect professionnel.
